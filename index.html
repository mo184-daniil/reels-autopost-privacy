Privacy Policy

Last updated: 24 June 2026

This application ("the App") is a personal automation tool operated by its owner. It uploads the owner's own short-form videos to the owner's own YouTube channel and generates titles and descriptions automatically. The App is not a public product and does not serve third-party end users.

Google / YouTube data we access

The App uses YouTube API Services. With the owner's explicit consent via Google OAuth 2.0, the App requests the https://www.googleapis.com/auth/youtube.upload scope. This scope is used solely to upload videos to the owner's own YouTube channel. The App does not read, collect, scrape, or republish any other user's data or content.

How we store and use data

  • OAuth tokens are stored securely on the owner's own private server and are used only to upload videos on the owner's behalf.
  • We do not sell, share, or transfer Google user data to any third party.
  • We do not use Google user data for advertising or any purpose other than uploading the owner's own videos.
  • Video files and generated text are processed only to create and upload the owner's own content.

YouTube API Services

By using the App you also agree to the YouTube Terms of Service. The App's use of information received from YouTube API Services adheres to the YouTube API Services Developer Policies. Handling of information is also governed by the Google Privacy Policy.

Revoking access

The owner can revoke the App's access at any time via the Google security settings page.

Data retention

OAuth tokens are retained only while the App is in use and are deleted when access is revoked.
